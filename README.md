# Passo a Passo para Execução

Siga os comandos abaixo no terminal do VSCode para configurar e rodar a aplicação.

## 1. Clonar o repositório

```bash
git clone https://github.com/kaiquehsp/AV4-TPll.git
```

## 2. Entrar na pasta
```bash
cd atlantis-spa
```

## 3. Instalação das dependências
```bash
npm install
```

## 4. Iniciar aplicação
```bash
npm run dev
```
